# Gesture-Based-Elevator-Control-System
Touch-free elevator control using Arduino and APDS9960 gesture sensor

 Description:
This project implements a touch-free elevator control system using hand gestures.
It improves hygiene by replacing physical buttons with gesture recognition.

Components Used:
- Arduino UNO
- APDS9960 Gesture Sensor
- OLED Display
- Motor Driver (for simulation)

Gestures Used:
- UP    → Move elevator up
- DOWN  → Move elevator down
- LEFT  → Open door
- RIGHT → Close door

Software:
- Arduino IDE
- Embedded C

 Applications:
- Elevators
- ATMs
- Vending Machines
